서울시 응급실 도착 소요시간 (2015, 2019)¶

import numpy as np
import pandas as pd
import seaborn as sns  

import matplotlib.pyplot as plt
from matplotlib import font_manager, rc 

font_name = font_manager.FontProperties(fname="c:/Windows/Fonts/malgun.ttf").get_name()
rc('font', family=font_name)

er = pd.read_excel('발병 후 응급실 도착 소요시간 현황 전국 2019.xlsx', encoding='utf-8')

er = er.drop(0)

sido = {'서울': '서울', 
        '부산': '서울 외', '대구': '서울 외', '인천': '서울 외', '광주': '서울 외', '대전': '서울 외', '울산': '서울 외', 
        '경기': '서울 외', '강원': '서울 외', '충북': '서울 외', '충남': '서울 외', '전북': '서울 외', '전남': '서울 외', 
        '경북': '서울 외', '경남': '서울 외', '제주': '서울 외'}

er['구분'] = er['지역별'].apply(lambda x : sido.get(x, '해당없음'))

er_pivot = pd.pivot_table(er, index='구분', aggfunc=np.sum) # '서울'과 '서울 외'로 나눔

del er_pivot['계']

er_pivot = er_pivot.transpose() # 행열 전환

# 소요시간 순서대로 정렬
er_pivot = er_pivot.reindex(index = ['30분 미만','30분~2시간', '2~4시간', '4~6시간', 
                                     '6~8시간', '8~12시간', '12~24시간', '24시간 이상'])

# 파이 차트 만들기
frequency = er_pivot['서울']
labels = er_pivot.index
explode = [0.05, 0.05, 0.05, 0.05, 0.05, 0.05, 0.05, 0.05]
colors = ['gold', 'yellow', 'dimgray', 'gray', 'darkgray', 'silver', 'lightgray', 'crimson']

# '2시간 미만'을 골든타임이라 가정하고 골드색으로 표시, 가장 오래 걸린 '24시간 이상'(붉은색)과 대조

plt.title("차트 1. 발병 후 응급실 도착 소요시간 (서울, 2019)")
plt.pie(frequency, 
        textprops={'fontsize': 13},
        explode=explode,
        labels=labels, 
        colors=colors, 
        autopct='%1.1f%%', 
        shadow=True, 
        startangle=90, 
        counterclock=False)
plt.axis('equal')
plt.savefig('hyun.png',dpi=300)

plt.show()

결론¶

    - 2019년 서울 기준으로, 골든타임(2시간으로 가정) 내 도착한 경우, 24시간 이상 소요된 경우가 각각 1/3 정도로 비슷한 수준임. (차트1)
    - 의외로 서울보다 서울 외 지역에서 '2시간 내 도착' 비율이 높고, '24시간 이상 소요'되는 비율이 낮음. (2015, 2019년 모두) (차트1~4)
    - 서울의 경우, 2015년 대비 2019년에 '2시간 내 도착' 비율은 감소했고, '24시간 이상 소요' 비율은 증가했음. (차트1,3)
    - 서울 외 지역의 경우 2015년과 2019년의 '2시간 내 도착', '24시간 이상 소요' 비율이 유사함. (차트2,4)
